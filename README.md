# Non-App Baby Presence Alert System

A Final Year Project focused on developing an IoT-based baby presence alert system using Arduino Uno. The project aims to provide an alert mechanism when a baby is detected in a potentially unsafe situation without requiring a mobile application.

## Features

* Baby presence detection using sensor-based monitoring.
* Audible alert system using a siren and relay module.
* Standalone operation without dependency on a mobile application.
* Prototype design and testing for functionality validation.

## Technologies Used

* Arduino Uno
* Embedded C/C++
* Sensors
* Relay Module
* Siren Alert System
* IoT Prototyping

## Project Scope

This project was developed as a prototype for academic purposes. Testing was conducted in a controlled environment to evaluate system functionality and alert response behavior.
